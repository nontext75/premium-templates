# Pla-tone Design Guide: "Technical Artistry"

## Core Manifesto
High-end Gundam specs and mechanical artistry presented as a premium technical exhibition. The aesthetic is "Spec-Ops precision" â€” cold, dark, and meticulously detailed.

## Typography & Scale
- **Primary Font**: `Inter` (Sans-serif)
- **Hierarchy**:
  - **Display Headers**: Massive, all-caps, `font-black`, `tracking-tighter`.
  - **Section Headers**: `text-2xl` to `text-3xl`, `font-bold`, `tracking-tight`.
  - **Labels / Data**: `font-mono`, `text-[10px]`, `tracking-widest`, `uppercase`.
  - **Body Copy**: `text-[14px]`, `font-light`, `leading-relaxed`.

## Color Palette
- **Primary (Background)**: `#0B0E14` (Deep Space Black)
- **Secondary (Accents)**: `#3B82F6` (Electric Blue) / `#EF4444` (Spec Red)
- **Text**: `#E2E8F0` (Steel White)
- **Muted**: `#1E293B` (Subtle UI borders / backgrounds)

## Layout & Components
- **Mechanical Grid**: Use modular blocks with technical data overlays.
- **Spec Overlays**: Use mono fonts for technical measurements and part numbers.
- **Interaction**: Glitch effects, scanning animations, and rigid, mechanical transitions.
