# Project: Gundam Plamodel Portfolio - GP.

## Project Overview
- **Name**: GP. (Gundam Plamodel Portfolio)
- **Purpose**: 건담 프라모델 커스터마이징 작업실 & 포트폴리오
- **Style**: 힙한 느낌의 다크 미니멀 디자인

## Site Features

### 1. Main Page (/)
- Hero: BUILD CREATE (포트폴리오 인트로)
- Collection: 완성된 작품 전시 (갤러리 그리드)
- Custom Service: 도색/커스텀 의뢰 안내
- Contact: 의뢰 문의 링크

### 2. Collection Page (/collection)
- 작품 그리드 (추후 구현)

### 3. About Page (/about)
- 작업실 소개 (추후 구현)

### 4. Contact Page (/contact)
- 의뢰 문의 (추후 구현)

## Design System
- **Primary Color**: #ff3b30 (빨간색 포인트)
- **Background Dark**: #1a1a2e (다크블루)
- **Background Light**: #ffffff (화이트)
- **Text**: #111111 / #ffffff
