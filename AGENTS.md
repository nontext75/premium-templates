# Project: Premium Next.js Templates

This repository contains a collection of premium Next.js templates, each designed for high-end digital experiences.

## Project Structure
- `commercial-brand/`: A template for commercial branding and corporate identity.
- `furniture-shop/`: An e-commerce oriented template specifically for interior and furniture design.
- `modern-agency/`: A creative agency template focusing on portfolio and services.
- `nside-template/`: A specialized template for interior design and architecture.

## Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS, Framer Motion, GSAP
- **Language**: TypeScript
- **CMS**: Supabase (Planned)

## Agent Instructions
- Follow the premium design language established in each template.
- Ensure all animations use GSAP/Framer Motion for smoothness.
- Maintain strict TypeScript types across the project.
- When modifying templates, preserve the "Block-based" architecture.
